# Phuket Go Task Map

Статус: справочный файл для владельца проекта.

Важно: эта папка и этот файл не являются техническим заданием.

Нельзя использовать этот файл как источник требований для разработки.

Правильные источники требований находятся в `docs/`.

Этот файл нужен только для понимания, на каком крупном этапе находится проект.

## Легенда

- `[done]` - этап выполнен.
- `[in progress]` - этап сейчас в работе.
- `[todo]` - этап еще не начат.
- `[later]` - этап на будущее, не для MVP.

## Как обновлять этот файл

Когда крупный этап завершен:

1. Меняем его статус на `[done]`.
2. Следующий этап меняем на `[in progress]`.
3. Не добавляем мелкие внутренние задачи.
4. Не используем этот файл как источник требований.
