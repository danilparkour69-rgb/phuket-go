# Monetization Options

Статус: [~] draft

| Модель | Как работает | Плюсы | Минусы | Подходит для MVP |
| --- | --- | --- | --- | --- |
| Обратная комиссия | Phuket Go получает комиссию от партнера, если услуга была оказана | Не нужна оплата на сайте, подходит для старта | Нужно контролировать факт оказания услуги | yes |
| Подписка партнеров | Партнер платит за доступ/размещение | Предсказуемый доход | Не приоритет | no |
| Реклама | Платные рекламные места | Можно добавить позже | Может ухудшить доверие на старте | no |
| Платное продвижение | Партнер платит за поднятие карточки | Подходит при большом каталоге | Не приоритет | no |
| Собственные услуги | Phuket Go продает свои услуги | Высокий контроль качества | Нужно операционное исполнение | future priority |
