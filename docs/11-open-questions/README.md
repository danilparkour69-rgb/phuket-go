# Этап 11. Открытые вопросы

## Цель

Зафиксировать, какие вопросы уже закрыты для локальной разработки MVP, а какие перенесены на публикацию или будущие этапы.

## Текущий статус

Критичных блокеров локальной разработки MVP нет.

Перед началом разработки остается один обязательный шаг: запросить у владельца проекта файл с инструкцией, прочитать его и только после этого писать код.

Подробный триаж:

- `open-questions-triage.md`

Ответы:

- `answers.md`

## Закрыто для локальной разработки

- Монетизация MVP: обратная комиссия после оказания услуги.
- География MVP: только Пхукет.
- Каталог MVP: управляемый каталог экскурсий.
- Оплата: не на сайте, пользователь оставляет заявку.
- Авторизация: Telegram, Google, email; SMS по телефону не используем.
- Личный кабинет: история заявок, статусы, баллы, первый уровень.
- Админка: заявки, пользователи, экскурсии, отзывы, статьи, комиссии, аналитика.
- Геймификация MVP: 100 баллов за регистрацию и первый уровень.
- Фото: не блокируют локальную разработку, берем `media-manifest.json` и `final/carousel`.
