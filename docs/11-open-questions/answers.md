# Answers

Статус: [~] draft

Здесь фиксируем ответы на критичные вопросы.

| Вопрос | Ответ | Влияние на MVP | Статус |
| --- | --- | --- | --- |
| Монетизация | Обратная комиссия с партнера/исполнителя после оказания услуги. Оплаты на сайте нет. | high | answered |
| География | MVP только Пхукет. В будущем нужно масштабирование на весь Таиланд и, возможно, другие регионы. | high | answered |
| Партнеры добавляют услуги сами? | В MVP нет. Карточки добавляет команда проекта/администратор. В будущем нужно заложить базу для партнерского добавления услуг. | high | answered |
| Каталог фиксированный или маркетплейс? | MVP ближе к управляемому каталогу с заявками. В будущем возможен полноценный маркетплейс с партнерскими кабинетами. | high | answered |
| Оплата внутри сайта? | Нет в MVP. Оплата происходит на стороне турагентства/партнера. Возможность будущей оплаты можно оставить архитектурно. | high | answered |
| Языки первой версии | Основной язык MVP - русский. Английский позже. Тайский/китайский пока не подтверждены. | high | partial |
| Способы авторизации | Регистрация входит в MVP. Выбираем бесплатные способы без SMS: Telegram, Google, email. Телефон через SMS не используем в MVP. | medium | answered |
| Состав личного кабинета | В MVP авторизованный пользователь видит историю заявок/бронирований с датами. Регистрация также нужна как база для будущей игры. | medium | answered |
| Состав админ-панели | Пользователи, заявки пользователя, общий список заявок, статусы заявок, история изменений, исполнитель, управление карточками, ручные отзывы. | high | answered |
| Геймификация в MVP | Минимально: регистрация, 100 стартовых баллов, первый уровень. Полноценную игру не делаем. | medium | answered |
| Хранение заявок | Временно Google Sheets, целевая база Postgres. | high | answered |
| Финальный статус заявки | Статус "Оказана" подтверждает партнер. | high | answered |
| Роли в админке | В MVP только владелец проекта, без менеджеров. | medium | answered |
| Комиссия MVP | Для экскурсий 100 бат с одного человека после оказания услуги. Для других категорий ставки будут добавляться позже. | high | answered |
| Цена услуги в заявке | Нужна. | medium | answered |
| Язык гида в заявке | Не нужен в Google Sheets для заявки. | low | answered |
| Район/отель клиента | Не нужен в MVP. | low | answered |
