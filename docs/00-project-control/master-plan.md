## Definition of Ready для начала кодинга

- [x] Есть утвержденный MVP.
- [x] Есть PRD или набор продуктовых требований.
- [x] Есть карта ролей и прав доступа.
- [x] Есть структура каталога услуг.
- [x] Есть схема обработки заявок.
- [x] Есть решение по оплатам: заявки без онлайн-оплаты, обратная комиссия.
- [x] Есть первичная архитектура: фронтенд, бэкенд, база, API, админка.
- [x] Есть модель данных на уровне сущностей.
- [x] Есть дорожная карта разработки по спринтам или этапам.
