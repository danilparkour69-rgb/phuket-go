# Paid Services Decision

Статус: актуально для MVP

## Решение

На текущем этапе Phuket Go не подключает платные сервисы, платные тарифы, платные облака, платные API, платежные провайдеры, аккаунты App Store / Google Play и любые другие платные внешние инструменты без отдельного решения пользователя.

## Что уже принято

- текущая цель - локальная разработка и локальная проверка MVP;
- онлайн-оплата не входит в MVP;
- production deployment пока не запускаем;
- mobile release и Telegram Mini App не входят в текущий этап;
- DigitalOcean, Yandex Cloud, платные Telegram/Google/сторонние интеграции не активируем автоматически.

## Что это значит для разработки

- можно делать только локальную архитектуру и бесплатные/встроенные части проекта;
- платные внешние шаги сначала выносятся пользователю как отдельное решение;
- если для следующего этапа понадобится платный ресурс, это считается точкой остановки и согласования.
