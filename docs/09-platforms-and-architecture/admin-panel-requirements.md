## MVP

- одна роль администратора: владелец проекта;
- список пользователей;
- карточка пользователя;
- список заявок пользователя;
- общий список заявок;
- просмотр деталей заявки;
- отображение статуса заявки;
- отображение исполнителя/партнера;
- история изменения статуса заявки;
- управление карточками услуг;
- ручное добавление отзывов;
- базовая аналитика по заявкам.

## Роли

- В MVP админкой пользуется только владелец проекта.
- Роль менеджера не нужна в MVP.
