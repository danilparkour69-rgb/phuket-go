# Data Migration Plan

Статус: [~] draft

## Цель

Перенести данные из Google Sheets в Postgres после того, как данные будут внесены и подготовлены.

## MVP

На первом этапе Google Sheets используется как временная база.

После заполнения и проверки данных выполняется перенос в Postgres.

## Что переносим

- заявки;
- экскурсии;
- партнеров/исполнителей;
- комиссии;
- пользователей, если они уже есть;
- отзывы;
- статьи, если они хранились во временной структуре.

## Важно

Структуру Google Sheets нужно делать так, чтобы ее можно было перенести в Postgres без переделки логики проекта.
