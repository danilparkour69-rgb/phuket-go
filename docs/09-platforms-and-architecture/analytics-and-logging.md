# Analytics And Logging

Статус: [~] draft

## Аналитика MVP

Нужно отслеживать:

- количество заявок;
- заявки по экскурсиям;
- заявки по партнерам;
- оказанные заявки;
- сумма комиссии.

## Логирование MVP

Нужно фиксировать:

- создание заявки;
- смену статуса заявки;
- кто сменил статус;
- действия администратора;
- действия партнера в Telegram-боте;
- ошибки маршрутизации;
- ошибки отправки уведомлений.
