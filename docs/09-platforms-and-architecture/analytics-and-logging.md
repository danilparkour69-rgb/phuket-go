# Analytics And Logging

Статус: [~] draft

## Аналитика MVP

Нужно отслеживать:

- количество заявок;
- заявки по экскурсиям;
- заявки по партнерам;
- оказанные заявки;
- сумма комиссии.

Расширенная продуктовая аналитика трафика, статей, источников, поведения и A/B-тестов пока зафиксирована как будущая идея в `../01-product-vision/product-ideas.md`.

## Логирование MVP

Нужно фиксировать:

- создание заявки;
- смену статуса заявки;
- кто сменил статус;
- действия администратора;
- действия партнера в Telegram-боте;
- ошибки маршрутизации;
- ошибки отправки уведомлений.
