# Skill: pishi-sokrashchai

## Что делает

Skill `pishi-sokrashchai` нужен для написания, переписывания и сокращения текстов.

Он помогает делать текст:

- понятным;
- коротким без потери смысла;
- полезным для читателя;
- основанным на фактах;
- без воды, клише и искусственных "продающих" фраз.

## Когда использовать

Использовать для:

- описаний экскурсий;
- SEO-страниц;
- статей;
- карточек услуг;
- текстов кнопок и интерфейса;
- писем;
- постов;
- инструкций;
- промптов;
- редактуры готовых текстов.

## Как применять в Phuket Go

Для проекта Phuket Go этот skill нужен, чтобы переписывать тексты экскурсий из источников.

Правило:

- факты, цены, маршрут и условия берем из источника;
- текст не копируем;
- переписываем так, чтобы он был понятным, живым, вовлекающим и помогал человеку самому принять решение о поездке.

## Триггеры

Skill включается, когда пользователь пишет:

- `пиши и сокращай`;
- `пиши, сокращай`;
- `пиши сокращай`;
- или просит написать, переписать, улучшить, сократить текст.
