# Russian Competitors Research Task

Статус: [ ] todo

## Цель

Собрать топ 10-20 русскоязычных конкурентов по экскурсиям на Пхукете.

## Что собрать по каждому конкуренту

- название;
- ссылка;
- какие экскурсии продают;
- как оформлена карточка экскурсии;
- есть ли цены;
- есть ли отзывы;
- есть ли оплата онлайн;
- как оставить заявку;
- есть ли Telegram/WhatsApp;
- сильные стороны;
- слабые стороны;
- что Phuket Go может сделать лучше.

## Где искать

- Google;
- Yandex;
- Telegram-каналы и чаты, если пользователь даст ссылки/названия/экспорт;
- выдача по запросам:
  - экскурсии Пхукет;
  - экскурсии на Пхукете;
  - Пхи-Пхи экскурсия Пхукет;
  - Симиланы экскурсия Пхукет;
  - русские экскурсии Пхукет;
  - русский гид Пхукет.

## Статус

Нужно выполнить отдельным исследованием.

В текущем проходе надежный список 10-20 локальных русскоязычных сайтов не подтвержден, поэтому нельзя записывать непроверенные домены как факты.
