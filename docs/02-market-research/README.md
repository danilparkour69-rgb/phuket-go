# Этап 2. Исследование конкурентов

## Цель

Понять рынок, конкурентов и свободные возможности до проектирования продукта.

## Группы конкурентов

- Русскоязычные агрегаторы экскурсий.
- Русские сервисы на Пхукете.
- Telegram-каталоги и чаты.
- Сайты локальных агентств.
- Англоязычные мировые маркетплейсы экскурсий.
- Сервисы аренды транспорта.
- Туристические super apps.
- Локальные приложения Таиланда.
