# Этап 10. MVP Roadmap

## Цель

Определить минимальную первую версию, которую реально запустить, протестировать и развивать.

## Принцип MVP

MVP должен доказать спрос, обработку заявок и партнерскую модель. Все, что не помогает этому, переносится в будущие версии.
