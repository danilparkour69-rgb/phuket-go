# Success Metrics

Статус: [~] draft

## Метрики спроса

- посетители сайта;
- просмотры страниц экскурсий;
- количество регистраций;
- количество пользователей, которые вернулись повторно.

## Метрики заявок

- количество заявок;
- сколько заявок взяли в работу;
- сколько заявок оказано;
- сколько заявок отклонено;
- какие экскурсии дают больше всего заявок;
- какие статьи приводят к заявкам.

## Метрики партнеров

- сколько партнеров реально обрабатывают заявки;
- заявки по партнерам;
- оказанные заявки по партнерам;
- отклоненные заявки по партнерам.

## SEO-метрики

- просмотры страниц экскурсий;
- просмотры статей;
- переходы из статей к экскурсиям;
- какие статьи приводят к заявкам.

## Финансовые метрики

- сумма комиссии;
- начисленная комиссия;
- оплаченная комиссия;
- долги партнеров.

## Контент и доверие

- количество добавленных отзывов;
- количество отзывов с фото;
- какие экскурсии получают больше отзывов.
