# MVP Scope

Статус: [~] draft

## Входит в MVP

- сайт;
- каталог экскурсий;
- отдельная страница каждой экскурсии;
- статьи;
- FAQ;
- контакты;
- условия бронирования;
- заявка без оплаты;
- Google Sheets как временное хранилище заявок;
- админка;
- Telegram-бот для партнеров;
- Telegram-бот для пользователей;
- регистрация;
- история заявок;
- баллы и первый уровень;
- отзывы после одобрения;
- аналитика заявок и комиссии.

## Не входит в MVP

- онлайн-оплата;
- партнерский кабинет;
- Telegram Mini App;
- Android/iPhone;
- недвижимость;
- байки;
- визы;
- документы;
- полноценная игра;
- мультиязычность;
- самостоятельное добавление услуг партнерами.

## Главная гипотеза

Русскоязычные туристы будут оставлять заявки на экскурсии через Phuket Go, если карточки оформлены удобно, есть доверие и связь через Telegram.
