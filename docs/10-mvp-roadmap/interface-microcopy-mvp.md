# Микротексты интерфейса MVP

Статус: [x] draft для будущей разработки

Документ фиксирует короткие тексты интерфейса Phuket Go: кнопки, статусы, сообщения формы заявки, уведомления пользователя, партнера и админа.

## Общий тон

- обращаемся на "вы";
- пишем коротко и понятно;
- не давим на человека;
- не обещаем то, что зависит от партнера, погоды, правил или доступности мест;
- объясняем следующий шаг.

## Главные кнопки сайта

| Место | Текст кнопки | Задача |
| --- | --- | --- |
| Первый экран | `Смотреть экскурсии` | Перейти к каталогу |
| Первый экран | `Оставить заявку` | Быстро открыть форму |
| Карточка экскурсии | `Оставить заявку` | Оставить заявку по конкретной экскурсии |
| Страница экскурсии | `Оставить заявку` | Открыть форму |
| Карточка экскурсии, будущий вариант | `Забронировать бесплатно` | Показать, что первый шаг без оплаты на сайте |
| Страница экскурсии, будущий вариант | `Бесплатное бронирование` | Снизить страх предоплаты, если условия подтверждены |
| Поддержка | `Задать вопрос` | Открыть помощь: AI для общих вопросов, менеджер для конкретной экскурсии |
| Финальный блок | `Подобрать экскурсию` | Помочь тем, кто не выбрал |
| Каталог | `Показать экскурсии` | Применить фильтр |
| Форма | `Отправить заявку` | Создать заявку |
| Личный кабинет | `Мои заявки` | Открыть историю |
| Личный кабинет | `Мои баллы` | Открыть баллы и уровень |

## Форма заявки

### Заголовок формы

```text
Оставить заявку
```

### Подзаголовок

```text
Заполните имя и телефон. Менеджер уточнит детали и подтвердит возможность поездки.
```

### Поля

| Поле | Название | Подсказка |
| --- | --- | --- |
| Имя | `Ваше имя` | `Как к вам обращаться` |
| Телефон | `Телефон` | `Для связи по заявке` |
| Telegram | `Telegram` | `Если удобнее написать вам в Telegram` |
| Дата | `Желаемая дата` | `Можно оставить пустым, если дата пока не выбрана` |
| Люди | `Количество людей` | `Взрослые и дети` |
| Комментарий | `Комментарий` | `Напишите важные пожелания или вопросы` |

Обязательные поля:

- имя;
- телефон.

### Текст перед отправкой

```text
Оплата на сайте не требуется. После заявки с вами свяжутся для подтверждения деталей.
```

### Пояснение к бесплатному бронированию

```text
Бронирование без оплаты на сайте. Менеджер подтвердит дату, наличие мест и условия перед оплатой.
```

Важно: формулировку про оплату не позднее чем за 24 часа можно использовать только после подтверждения правил с партнерами по конкретным экскурсиям.

### Поддержка

```text
Задайте вопрос
```

```text
На общие вопросы ответит AI-ассистент. Если вопрос касается конкретной экскурсии, даты, цены или оплаты, подключим менеджера.
```

### Успешная отправка на сайте

```text
Спасибо за обращение!

Менеджер свяжется с вами в ближайшее время.
```

После этого пользователь выбирает удобный способ связи.

### Выбор канала связи после заявки

Каналы:

- Telegram;
- WhatsApp;
- Max;
- телефон.

Предзаполненное сообщение для мессенджера:

```text
Здравствуйте, хочу записаться на экскурсию.

Экскурсия: {excursion_title}
Дата: {preferred_date}
Количество людей: {people_count}
```

Если дата или количество людей не выбраны, эти строки не добавляем.

### Ошибка отправки

```text
Не удалось отправить заявку.

Проверьте данные и попробуйте еще раз. Если ошибка повторится, напишите нам в Telegram.
```

### Ошибка обязательного поля

```text
Заполните это поле.
```

### Ошибка телефона

```text
Проверьте номер телефона.
```

## Статусы заявки для пользователя

| Системный статус | Текст для пользователя | Смысл |
| --- | --- | --- |
| `new` | `Заявка отправлена` | Заявка создана |
| `waiting_partner` | `Ожидаем подтверждения` | Заявка передана исполнителю |
| `accepted` | `Заявка взята в работу` | Исполнитель принял заявку |
| `declined` | `Нужна замена или уточнение` | Исполнитель отказался или мест нет |
| `completed` | `Поездка состоялась` | Услуга оказана |
| `cancelled` | `Заявка отменена` | Заявка закрыта без оказания услуги |

## Сообщения пользователю

### После отправки заявки

```text
Спасибо за обращение!

Менеджер свяжется с вами в ближайшее время.
```

### Ожидаем подтверждения

```text
Ожидаем подтверждения.

Заявка #{public_number} передана исполнителю. Мы сообщим, когда появится ответ.
```

### Заявка взята в работу

```text
Заявка взята в работу.

Исполнитель свяжется с вами, чтобы подтвердить детали поездки.
```

### Исполнитель отклонил заявку

```text
По этой заявке нужно уточнение.

Исполнитель не смог сразу принять заявку. Администратор Phuket Go проверит ситуацию и при необходимости предложит другой вариант.
```

### Поездка состоялась

```text
Поездка отмечена как состоявшаяся.

Спасибо, что выбрали Phuket Go.
```

Для авторизованного пользователя дополнительно:

```text
Баллы будут начислены после финальной проверки заявки.
```

### Заявка отменена

```text
Заявка отменена.

Если хотите выбрать другую экскурсию, мы поможем с подбором.
```

## Сообщения партнеру

### Новая заявка

```text
Новая заявка Phuket Go

Заявка: #{public_number}
Экскурсия: {excursion_title}
Дата: {preferred_date}
Количество людей: {people_count}

Клиент: {customer_name}
Телефон: {customer_phone}
Telegram: {customer_telegram}

Комментарий: {customer_comment}
```

Кнопки:

- `Взять в работу`;
- `Отклонить`;
- `Связаться с клиентом`.

### Партнер взял заявку

```text
Заявка #{public_number} взята в работу.

Свяжитесь с клиентом и подтвердите детали.
```

### Партнер отклонил заявку

```text
Заявка #{public_number} отклонена.

Администратор увидит это и решит, что делать дальше.
```

### Партнер отметил оказание услуги

```text
Заявка #{public_number} отмечена как оказанная.

Комиссия будет учтена в месячном расчете.
```

```text

```

## Личный кабинет

### Пустая история заявок

```text
У вас пока нет заявок.
```

Кнопка:

```text
Смотреть экскурсии
```

### Карточка заявки

Показываем:

- номер заявки;
- экскурсию;
- дату;
- количество людей;
- статус;
- баллы, если поездка состоялась и пользователь авторизован.

### Баллы

```text
Ваш уровень: {level_name}
Баллы: {points}
```

Для будущего начисления:

```text
+{points} баллов после подтвержденной поездки
```

## Пустые состояния

### Нет экскурсий по фильтру

```text
По этим фильтрам пока ничего не найдено.

Попробуйте изменить категорию или цену.
```

### Нет отзывов

```text
Отзывы по этой экскурсии появятся позже.
```

### Нет статей

```text
Полезные статьи скоро появятся.
```

## Предупреждения

### Цена

```text
от {price_thb} ฿
примерно {price_rub} ₽ по текущему курсу
```

Для будущей английской версии:

```text
from {price_thb} ฿
about ${price_usd} at the current exchange rate
```

### Оплата не на сайте

```text
Оплата проходит у партнера после подтверждения деталей.
```

### Уточнение условий

```text
Некоторые условия зависят от даты, погоды и доступности мест. Менеджер уточнит детали перед подтверждением.
```

## Правило для будущей разработки

Все статусы и тексты уведомлений должны быть едиными для:

- сайта;
- личного кабинета;
- админки;
- Telegram-ботов;
- будущего Telegram Mini App.

Если текст меняется, его нужно менять в одном месте логики, а не отдельно в каждом интерфейсе.
