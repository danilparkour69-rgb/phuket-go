# Карта данных для главной и каталога MVP

Статус: [x] draft для будущей разработки

Кодинг проекта пока не начинаем.

Документ фиксирует, какие экскурсии показывать на главной, как разложить каталог по категориям и какие статьи связывать с какими экскурсиями.

## Главный принцип

На главной и в каталоге человек должен выбирать не “товар”, а желаемый день:

- острова и море;
- красивые виды;
- активный отдых;
- вечерние эмоции;
- семейный формат;
- премиум-отдых.
