# User Roles

Статус: [~] draft

## Гость

Неавторизованный пользователь.

Может:

- смотреть экскурсии;
- читать статьи;
- оставлять заявки без регистрации.

## Пользователь

Авторизованный пользователь.

Может:

- смотреть экскурсии;
- читать статьи;
- оставлять заявки;
- видеть историю своих заявок;
- видеть баллы;
- видеть первый уровень.

## Партнер

Исполнитель услуги.

В MVP не имеет личного кабинета.

Работает через Telegram-бот:

- получает заявку;
- нажимает "Взята в работу";
- нажимает "Отклонено";
- подтверждает "Оказана".

## Администратор

В MVP администратор один: владелец проекта.

Видит и управляет:

- экскурсиями;
- заявками;
- пользователями;
- отзывами;
- статьями;
- статусами заявок;
- исполнителями внутри админки.

Отзывы публикуются только после личного одобрения администратора.
