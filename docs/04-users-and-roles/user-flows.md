# User Flows

Статус: [~] draft

## Гость

1. Заходит на сайт.
2. Смотрит экскурсии.
3. Читает статьи.
4. Открывает карточку экскурсии.
5. Нажимает "Оставить заявку".
6. Заполняет форму.
7. Отправляет заявку без регистрации.
8. Дальше получает связь через Telegram или телефон.

## Пользователь

1. Авторизуется любым доступным способом.
2. Смотрит экскурсии и статьи.
3. Оставляет заявку.
4. Видит заявку в истории.
5. Видит баллы.
6. Видит первый уровень.

## Партнер

1. Получает заявку в Telegram-бот.
2. Нажимает "Взята в работу" или "Отклонено".
3. Если взял в работу, связывается с клиентом.
4. После оказания услуги подтверждает статус "Оказана".

## Администратор

1. Заходит в админ-панель.
2. Видит экскурсии.
3. Видит заявки.
4. Видит пользователей.
5. Видит исполнителей.
6. Управляет отзывами.
7. Публикует отзывы только после личного одобрения.
8. Управляет статьями.
