# Account Requirements

Статус: [~] draft

## Авторизация

Регистрация входит в MVP, но заявка должна отправляться и без регистрации.

В MVP выбираем бесплатные способы входа без SMS:

- Telegram;
- Google;
- email;

Позже можно рассмотреть:

- телефон, если SMS будут экономически оправданы;
- ВКонтакте;
- другие способы входа.

## Личный кабинет пользователя

Минимальная функция MVP: авторизованный пользователь видит, на какие даты и какие услуги он забронировал/оставил заявки.

Также пользователь видит:

- баллы;
- первый уровень.

Полноценная игровая механика не входит в MVP, но регистрация должна закладывать базу для будущей игры.

## Кабинет партнера

В MVP кабинета партнера нет.

Партнер работает через Telegram-бот.

## Админ-панель

В MVP админ один: владелец проекта.

Админ видит:

- экскурсии;
- заявки;
- пользователей;
- отзывы;
- статьи;
- исполнителей;
- статусы заявок.

Отзывы публикуются только после личного одобрения администратора.
