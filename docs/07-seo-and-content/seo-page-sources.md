# SEO Page Sources

Статус: [~] draft

## Задача

Понять, из каких источников брать информацию для структуры первых SEO-страниц.

## Источники для структуры SEO-страниц

- страницы конкурентов по конкретным экскурсиям;
- Tripster;
- Sputnik8;
- Klook;
- GetYourGuide;
- русскоязычные сайты агентств Пхукета;
- статьи конкурентов по островам, храмам, районам и правилам Таиланда;
- вопросы туристов из Telegram-чатов;
- вопросы туристов из комментариев и отзывов;
- поисковые подсказки Google/Yandex;
- Yandex Wordstat, если будет доступ;
- Google Search Console после запуска сайта.

## Что нужно получить из источников

- какие заголовки используют конкуренты;
- какие вопросы задают туристы;
- какие блоки повторяются на успешных страницах;
- какие слова используют люди при поиске;
- какие страхи и возражения нужно закрыть;
- какие экскурсии связывать со статьями.

## Что дать Codex

Пользователь может дать:

- ссылки на страницы конкурентов;
- ссылки на экскурсии;
- список Telegram-чатов/каналов;
- выгрузку вопросов из чатов;
- книгу для создания skill по стилю текстов.

## Будущий skill для текстов

Пользователь даст книгу. На ее основе нужно сделать skill для переписывания текстов экскурсий:

- интересно;
- вовлекающе;
- продает идею поездки;
- человек сам принимает решение;
- без прямого давления;
- с сокращением лишнего текста.
