# SEO Structure

Статус: [~] draft

## URL-структура

- отдельная страница для каждой экскурсии;
- одна общая страница каталога экскурсий с фильтрами;
- отдельные страницы направлений в MVP не делаем;
- статьи имеют отдельные страницы;
- FAQ, полезные советы, условия бронирования и контакты имеют отдельные страницы.

Примеры:

- `/excursions`
- `/excursions/phi-phi-islands`
- `/articles/kak-priehat-v-thailand`
- `/faq`
- `/booking-terms`
- `/contacts`

## Шаблоны страниц

- страница экскурсии;
- общая страница каталога экскурсий;
- статья;
- FAQ;
- полезные советы;
- условия бронирования;
- контакты.

## Мультиязычность

MVP только на русском языке.

Английский язык позже.

## Внутренняя перелинковка

- статьи должны ссылаться на подходящие экскурсии;
- карточки экскурсий могут ссылаться на релевантные статьи;
- пример: статья про острова Пхи-Пхи содержит блок с карточкой экскурсии на Пхи-Пхи.
