# Этап 7. SEO и контент

## Цель

Заложить SEO-структуру до разработки, чтобы продукт мог получать органический трафик.

## Принцип контента

Phuket Go продает эмоции, а не экскурсии.

В текстах нужно показывать не только маршрут и цену, а то, какой день человек получит:

- что он почувствует;
- какие моменты запомнит;
- почему это удобно и спокойно;
- какие сомнения снимает Phuket Go;
- почему заявку безопаснее оставить через понятный сервис, а не искать посты в Telegram.

## Типы страниц

- Страница каждой услуги.
- Страница категории.
- Страница пляжа.
- Страница острова.
- Страница района.
- Статья.
- Инструкция.
- FAQ.
- Путеводитель.
- Бесплатный гайд.
