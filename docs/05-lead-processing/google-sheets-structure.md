## Зачем нужна таблица

Google Sheets временно хранит заявки до перехода на Postgres.

## Структура

В MVP используем один лист Google Sheets.

Одна строка = одна заявка.

В этой строке хранится вся информация: клиент, услуга, партнер, статус, цена, комиссия, телефон, Telegram и заметки.

## Лист: Заявки

Колонки:

| Колонка | Что означает |
| --- | --- |
| ID заявки | Уникальный номер заявки |
| Дата создания | Когда заявка пришла |
| Статус | Новая, Взята в работу, Отклонена, Оказана, Отменена |
| Название услуги | Какая экскурсия или услуга выбрана |
| Категория услуги | Экскурсия, морской тур, аренда байка, обмен денег и т.д. |
| Дата услуги | На какую дату хочет пользователь |
| Количество людей | Сколько человек участвует |
| Цена услуги | Цена услуги для пользователя |
| Имя клиента | Имя пользователя |
| Телефон клиента | Телефон пользователя |
| Telegram клиента | Telegram пользователя, если указал |
| Комментарий клиента | Дополнительная информация |
| Авторизован | Да/нет |
| ID пользователя | Заполняется, если пользователь авторизован |
| Исполнитель | Кто оказывает услугу |
| Telegram исполнителя | Куда отправлять заявку исполнителю |
| Комиссия за человека | По умолчанию 100 бат для экскурсий |
| Итоговая комиссия | Комиссия за человека * количество людей |
| Статус комиссии | Не начислена, начислена, оплачена |
| Заметка админа | Внутренний комментарий |
| Кем создана | Пользователь, гость, админ |
| Когда взята в работу | Время смены статуса |
| Когда отклонена | Время смены статуса |
| Когда оказана | Время смены статуса |
| Кто менял статус | Партнер или админ |

## История в админке

В админке должно быть видно:

- кем создана заявка;
- когда создана заявка;
- текущий статус;
- когда статус сменился на "Взята в работу";
- когда статус сменился на "Отклонена";
- когда статус сменился на "Оказана";
- кто менял статус.

## Аналитика MVP

- количество заявок;
- заявки по экскурсиям;
- заявки по партнерам;
- оказанные заявки;
- сумма комиссии.

## Не добавляем в MVP

- язык гида;
- район/отель клиента;
- рекламные/маркетинговые метки.

## Не делаем в MVP

- отдельный лист "Услуги";
- отдельный лист "Партнеры";
- отдельный лист "Комиссии".

## Открытые вопросы

- Для каких категорий комиссия отличается от 100 бат за человека.
