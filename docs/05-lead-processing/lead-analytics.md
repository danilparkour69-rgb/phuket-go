# Lead Analytics

Статус: [~] draft

## Аналитика MVP

В админке нужно видеть:

- количество заявок;
- заявки по экскурсиям;
- заявки по партнерам;
- оказанные заявки;
- сумма комиссии.

## Зачем

- понимать, какие экскурсии дают больше заявок;
- видеть, какие партнеры обрабатывают заявки;
- считать фактическую комиссию после оказания услуги.
