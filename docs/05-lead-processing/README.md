# Этап 5. Обработка заявок

## Цель

Описать полный путь заявки от пользователя до партнера и обратно.

## Предварительная схема

Пользователь выбирает услугу, оставляет заявку, система определяет категорию и партнера, отправляет заявку партнеру и дублирует владельцу платформы. Партнер меняет статус, пользователь видит обновление.

## Статусы заявки

- Новая.
- Получена.
- В работе.
- Связались.
- Забронировано.
- Выполнено.
- Отменена.
