# Lead History

Статус: [~] draft

## История заявки

В админке должно быть видно:

- кем создана заявка;
- когда создана заявка;
- текущий статус заявки;
- когда заявка взята в работу;
- когда заявка отклонена;
- когда заявка оказана;
- кто менял статус.

## Пользовательский статус

Пользователь видит статус заявки:

- в личном кабинете, если авторизован;
- через отдельный Telegram-бот для пользователей.
