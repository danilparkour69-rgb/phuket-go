## Пользователь

- Подтверждение, что заявка отправлена.
- После отправки заявки пользователь видит следующий экран выбора места для связи: Telegram, WhatsApp или Max.
- Уведомление при смене статуса заявки.

## Партнер

- Новая заявка по его услуге.
- Данные заявки: имя, телефон, Telegram, выбранный канал связи, дата, количество людей, комментарий.

## Администратор

- Все новые заявки.
- Ошибки маршрутизации.
- Заявки без назначенного партнера.

## MVP-уведомления

- пользователю после отправки заявки;
- партнеру повторное уведомление, если заявка долго не обработана;
- партнеру при новой заявке;
- админу при новой заявке;
- пользователю при смене статуса заявки.
