## Пользователь

- Подтверждение, что заявка отправлена.
- Уведомление при смене статуса заявки.

## Партнер

- Новая заявка по его услуге.
- Данные заявки: имя, телефон, Telegram, дата, количество людей, комментарий.

## Администратор

- Все новые заявки.
- Ошибки маршрутизации.
- Заявки без назначенного партнера.

## MVP-уведомления

- пользователю после отправки заявки;
- партнеру при новой заявке;
- админу при новой заявке;
- пользователю при смене статуса заявки.
