# Image Regeneration Briefs

Статус: [~] рабочий список на добор и перегенерацию

Дата: 2026-06-28

## Правило

Эти изображения нужны для направлений, где после чистки не хватает горизонтальных фото для верхней карусели.

Общие требования:

- формат: горизонтальный;
- назначение: верхняя карусель;
- стиль: реалистично, живо, красиво, без ощущения стока;
- без крупных узнаваемых лиц людей;
- без логотипов, брендов и неподтвержденных судов/площадок;
- не выдавать AI-фото за реальное фото конкретной экскурсии;
- фото должно продавать мечту, воспоминание и ощущение, а не просто показывать объект.

## 001 Слоны / Mantra Spa / Самет Нангше

Нужно: 1 горизонтальное фото.

Роль: `carousel_emotion`.

Кадр: этичное взаимодействие со слонами без катания. Подойдет кормление, наблюдение или спокойная прогулка рядом со слоном.

Эмоция: забота, доверие, мягкий контакт с природой.

Запреты:

- не показывать катание на слонах;
- не показывать цепи, крюки, жесткий контакт;
- не делать крупные лица людей главным объектом.

## Следующее действие

После ручной проверки оставшихся фото:

1. Удалить финальный список неподходящих фото.
2. По этому файлу подготовить генерацию или запрос партнеру.
3. После появления новых фото загрузить их в папки направлений.
4. Пометить тип источника: `partner`, `ai_generated`, `ai_enhanced` или другой выбранный тип.
