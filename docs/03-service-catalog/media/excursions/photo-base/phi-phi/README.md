## Как используем дальше

Эти файлы нужны как рабочая база:

- выбрать сильные визуальные сцены;
- понять, какие фото нужны для карусели;
- понять, какие фото нужны по текстовым блокам;
- подготовить задания на AI-генерацию новых изображений;
- заменить временные картинки перед публикацией.

К публикации будут подготовлены новые изображения.
