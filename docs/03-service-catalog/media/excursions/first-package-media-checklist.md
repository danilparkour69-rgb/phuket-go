Статус: [~] рабочий список

## Цель

Подготовить локальные фото для первого пакета экскурсий, чтобы карточки можно было переводить из `needs_media` в `ready_for_copy`.

## Правило

Для каждой экскурсии нужна отдельная папка:

`docs/03-service-catalog/media/excursions/ID-slug/`

Минимум для MVP:

- `cover.jpg` - обложка карточки;
- `gallery-01.jpg`;
- `gallery-02.jpg`;
- `gallery-03.jpg`;
- `source.txt` - источник, права, тип фото.

Тип фото в `source.txt`:

- `real`;
- `ai_enhanced`;
- `ai_generated`.

- Не оставлять в карточках технические ссылки `/file/...`.

- Не выдавать AI-фото за реальное фото экскурсии.
- Не использовать фото без понятного источника в финальной публикации.
