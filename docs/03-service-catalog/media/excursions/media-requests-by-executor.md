# Media Requests By Executor

Статус: [x] тексты запросов готовы

Дата: 2026-06-28

## Цель

Подготовить отдельные запросы на фото по группам исполнителей, чтобы не смешивать обычные экскурсии, дайвинг, шоу, яхты и активности.

## Общие требования к фото

Просим:

- реальные фото с экскурсий или площадок;
- горизонтальные фото для обложек;
- все доступные подходящие фото для галереи по каждой экскурсии;
- фото без сильных водяных знаков;
- фото без чужих логотипов, если нет разрешения;
- фото в хорошем качестве;
- отдельно подтвердить право использовать фото на сайте, в рекламе и соцсетях.

Внутри проекта каждое фото будет отмечено как:

- `real` - реальное фото;
- `ai_enhanced` - реальное фото, улучшенное нейросетью;
- `ai_generated` - полностью сгенерированное изображение.

AI-фото нельзя выдавать за реальное фото конкретной экскурсии.

## 6. Внутреннее правило после получения фото

Когда фото получены:

1. Сохранить в нужную папку экскурсии.
2. Назвать файлы:
   - `cover.jpg`;
   - `gallery-01.jpg`;
   - `gallery-02.jpg`;
   - `gallery-03.jpg`.
3. Обновить `media-manifest.json` конкретной экскурсии.
4. Обновить папки `original/`, `final/carousel` и `final/content`.
6. Если фото временное или из AI, явно указать тип.
