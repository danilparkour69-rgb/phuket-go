# Partner Photo Selection Index

Дата: 2026-06-28

## Задача

Рабочий индекс для отбора фото из карточек из базы фото.

Сначала сохраняем для каждой экскурсии все доступные фото по направлению, кроме:

- фото из отзывов.

После этого выбираем:

- все подходящие горизонтальные фото хорошего качества для первой карусели;
- фото для блоков внутри страницы;
- фото, которые нужно заменить или пересобрать позже.

## Где смотреть фото

Галерея ручной проверки:

- `docs/03-service-catalog/media/excursions/photo-base-review-gallery.html`

Папка с исходными локальными референсами:

- `docs/03-service-catalog/media/excursions/photo-base/`

## Правило отбора

Выбираем не просто красивые картинки, а фото, которые помогают человеку представить мечту, маршрут и ощущение дня.

Карусель должна быстро отвечать:

- куда человек попадет;
- что увидит;
- на чем поедет;
- что будет делать;
- какое воспоминание получит.

## Как заполнять после отбора

После ручного просмотра для каждой экскурсии добавляем:

- `cover`: имя выбранного файла;
- `carousel`: все подходящие файлы для карусели;
- `inline`: какие фото подходят внутрь текста;
- `replace`: какие фото слабые или повторяются;
- `notes`: что запросить у партнера или подготовить через AI.
