# Media Quality Audit

Статус: [~] рабочий аудит

Дата: 2026-06-28

## Цель

Понять, какие фото можно использовать в MVP, какие нужно добрать, а какие лучше улучшить через апскейл или заменить.

## Минимальный стандарт MVP

- Обложка: желательно от 1200 px по ширине.
- Галерея: желательно от 1000 px по ширине.
- Для карточки экскурсии: минимум 1 обложка + 3 фото галереи.
- Если фото меньше 1000 px по ширине, ставим `needs_upscale_or_replace`.

## Решение по источникам

Исходные страницы частичных карточек проверены. Дополнительных фото там не хватает.

Дальше работаем так:

1. Сначала запрашиваем реальные фото у партнера.
2. Если партнер не даст фото, подбираем разрешенные фото из открытых или покупных источников.
3. Если фото есть, но слабого качества, делаем `ai_enhanced`.
4. Если фото нет, допускаем `ai_generated`, но обязательно помечаем это в `source.txt`.

Подробный план: `missing-media-plan.md`.
