## Что требует внимания

### Временные Pexels-фото

Pexels-фото нельзя считать финальными партнерскими материалами. Для них нужно проверить:

- подходит ли фото по смыслу конкретной экскурсии;
- не показывает ли фото другую локацию как обещание;
- не выглядит ли фото случайной картинкой, не связанной с текстом;
- можно ли заменить фото партнерскими материалами;
- корректно ли указаны источник и права.
