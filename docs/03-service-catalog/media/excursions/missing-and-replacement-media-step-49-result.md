## Цель

Зафиксировать, какие фото нужно заменить или добрать после проверки дублей и слабого качества.

Структура будущей страницы экскурсии:

1. Сначала идет карусель из всех подходящих горизонтальных фото хорошего качества.
2. Дальше по тексту идут смысловые фото: транспорт, маршрут, активность, включения, безопасность, эмоция.

Сайты Sputnik8 и Tripster используем как референсы по сценам и подаче. Финальные фото для Phuket Go готовим отдельно.

## Правило карусели

Для каждой экскурсии карусель должна продавать мечту и быстро объяснять, куда человек попадет.

Типовые роли фото для карусели:

| Роль | Что показывает |
| --- | --- |
| `cover_emotion` | Самая сильная мечта: остров, вид, вода, закат, джунгли, шоу или активность |
| `route_place` | Главное место маршрута |
| `transport_or_format` | Катер, яхта, паром, машина, квадроцикл, лодка, площадка |
| `activity` | Что человек будет делать |
| `detail_or_trust` | Еда, оборудование, безопасность, гид, группа, атмосфера |

В карусель идут все подходящие фото направления.

## Правило фото внутри страницы

После карусели фото идут не случайно, а рядом с текстом:

| Блок страницы | Фото |

| Кому подойдет | спокойный отдых, семья, активность, романтика |

## Где брать замену

Порядок:

1. Партнерские фото или официальные материалы исполнителя.
2. Собственные фото Phuket Go.
3. Новые согласованные изображения по фотосценарию.
4. AI-generated с нуля, если реального фото нет.
5. AI-enhanced только для фото, на которые у нас есть право.

Sputnik8 и Tripster:

- использовать как визуальный ориентир по сценам;
- смотреть, какие фото ставят в карусель;
- смотреть, какие фото идут рядом с описанием;
- не ставить изображения конкурентов как финальные материалы Phuket Go.

## Что запросить у партнера

Нужно попросить по каждой проблемной экскурсии:

- 1 сильное горизонтальное фото для обложки;
- все подходящие горизонтальные фото хорошего качества для карусели;
- 3-7 дополнительных фото по тексту;
- фото без водяных знаков и чужих логотипов;
- подтверждение, что фото можно использовать на сайте, в рекламе и соцсетях.
