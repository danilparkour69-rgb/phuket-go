## MVP-категории

MVP стартует как управляемый каталог: карточки создает команда Phuket Go, а не партнеры.

В MVP оставляем только экскурсии.

Экскурсии могут быть разных направлений:

- острова;
- морские прогулки;
- прогулки по Пхукету;
- храмы;
- обзорные экскурсии;
- другие экскурсионные направления.

Точная база конкретных туров и услуг будет предоставлена позже. Отдельная задача MVP-подготовки - собрать и нормализовать базу экскурсий, вариантов, цен, условий, фотографий и партнеров.

Недвижимость, визы, документы, SIM-карты, медицина, обмен денег, аренда транспорта и другие направления остаются в будущей архитектуре, но не входят в MVP-каталог.

## Базы по категориям

Для каждой крупной категории в будущем нужна отдельная база:

- база экскурсий;
- база байков;
- база недвижимости;
- база документов/виз;
- база других категорий.

В MVP работаем только с базой экскурсий.
