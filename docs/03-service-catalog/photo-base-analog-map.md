## Цель

Для каждой экскурсии Phuket Go найти похожую карточку из базы фото, чтобы дальше брать фото только из основной карточки экскурсии.

Правило:

- берем фото только из основной карточки экскурсии;
- фото из отзывов не берем;
- если точного аналога нет, фиксируем ближайший аналог и пометку `нужно добрать точную ссылку`;
- к публикации будут подготовлены новые изображения.
