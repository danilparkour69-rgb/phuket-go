# Partner Questions

Статус: [~] получены частичные ответы партнера, условия MVP зафиксированы осторожными формулировками

Цель: получить недостающие данные по экскурсиям, чтобы обновить базу Phuket Go и не обещать пользователям то, что не подтверждено.

Главный принцип: мы продаем эмоции, а не экскурсии, но факты должны быть точными.

## Как использовать

1. Зафиксировать полученные ответы.
2. Отдельно пройти критичные вопросы по конкретным экскурсиям.
3. После ответов по ценам обновить карточки экскурсий.
4. Только после этого писать публичные SEO-тексты.

- Срок отмены без штрафа уточняется лично при бронировании.
- При плохой погоде: перенос или возврат.
- Решение о переносе принимает оператор, туристов оповещают.
- При отмене из-за погоды деньги возвращаются.
- Перенос даты без штрафа: нужно уточнить.
- Если турист опоздал на трансфер, его ждут. Если опоздание около 2 часов - отмена без возврата средств.

### Что включено

- Трансфер, питание, вода, фрукты, билеты, национальный парк, оборудование, полотенца, шкафчики включены там, где это предусмотрено программой.
- Где национальный парк оплачивается отдельно - уточнять лично при бронировании.
- Где входные билеты оплачиваются отдельно - уточнять лично при бронировании.
- Обязательные депозиты - не уточнено.
- Платные фото/видео - уточнять лично при бронировании.

### Ограничения

- Ограничения для беременных уточняются лично при бронировании.
- Ограничения для детей до 1 года уточняются лично при бронировании.
- Что значит "пожилым нельзя" - уточняется лично при бронировании.
- Ограничения по хроническим заболеваниям уточняются лично при бронировании.
