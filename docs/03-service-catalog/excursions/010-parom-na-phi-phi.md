## Коротко

Направление: Водный транспорт / трансфер.

Ключевые факты из источника:

- Как добраться до Пхи-Пхи из Пхукета? Самый простой способ посетить острова Пхи-Пхи - купить билеты на паром. Паром отправляется из Пхукета с пирса Rassada Pier и причаливают в порту Тонсай. Билет в обратную сторону продается с открытой датой, длительность вашего прибывания на острове зависит только от вас. Также вы можете добраться до Пхи-Пхи на скоростном катере
- билет на паром в обе стороны с открытой датой - 1200 Bath c человека.
- билет на паром в одну сторону - 900 Bath.
- Бесплатный трансфер с районов Ката, Карон, Патонг, Олд Таун
- Пхукет (пирс Рассада) - Пхи-Пхи (пирс Тонсай)
- Пхи-Пхи (пирс Тонсай) - Пхукет (пирс Рассада)
- Пхи-Пхи (пирс Тонсай) - Краби (пирс Клонг)
- Пхи-Пхи (пирс Тонсай) - Ао Нанг (пирс Пхара)
