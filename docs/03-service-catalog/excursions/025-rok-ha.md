## Коротко

Направление: Морская экскурсия / премиум.

Ключевые факты из источника:

- Андаманское море богато на красивые острова. Одни из них обитаемы, на другие проводятся экскурсии с Пхукета, третьи до сих пор не открыты для массового туризма. Крошечный архипелаг Рок и Ха расположился намного южнее Пхукета, островов Пхи Пхи и Ланты. Доехать сюда можно с экскурсией на скоростном катере. Путь неблизкий — с Пхукета катер едет около полутора часов. Зато острова стоят того, чтобы их посетить. Особенно искушённым путешественникам, которые уже всё видели и везде были.
- Тур проводится только по средам и субботам
- Программа тура может быть изменена в зависимости от погодных условий. Решение об изменении программы принимает компания проводящая тур исходя из соображений безопасности.
- Беременным, детям до 1 года, лицам в период обострения хронических заболеваний и с заболеваниями опорно-двигательного аппарата запрещается посещение экскурсии.
- Тур в сопровождении русского гида проводится только по субботам
- 3000 / 2800 baht - (взрослый/детский) - с одного человека
- В стоимость включено: Трансфер, Спасательные жилеты, Страховка, Прохладительные напитки, Фрукты, Обед, Маска и трубка для снорклинга
- Взять с собой: Купальный костюм, Полотенце, Головной убор, Солнцезащитное средство, Солнечные очки, Фотоаппарат, Видеокамера, Деньги на карманные расходы
