## Главное правило публикации

На публичный сайт нельзя выводить:

- `уточнить`;
- `черновик`;
- `блок не найден`;
- `по источнику`;
- `нужно спросить партнера`;
- внутренние комментарии про обработку;
- непроверенные обещания;
- условия, которые зависят от партнера, без пометки `уточняется перед бронированием`.

Подробные безопасные формулировки по условиям зафиксированы в:

- `conditions-step-41-result.md`

Если факт не подтвержден, его можно писать только мягко:

- `условия уточняются перед бронированием`;
- `точное время зависит от района отеля`;
- `итоговая цена подтверждается перед оплатой`;
- `ограничения нужно уточнить перед бронированием`;
- `программа может меняться из-за погоды и решения оператора`.

## Что уже подтверждено общим ответом партнера

Эти ответы можно использовать как базу, но для конкретной экскурсии все равно оставлять осторожную формулировку.

### Цены

- Цены актуальны сейчас.
- К сезону цены могут измениться.
- Возможное изменение: сентябрь-октябрь.
- Сезонных изменений сейчас нет.
- Детский тариф уточняется лично при бронировании.
- Цена для детей до 1 года уточняется лично при бронировании.
- Доплат за праздничные даты нет.

### Трансфер

- Районы трансфера уточняются при бронировании.
- По ответу партнера, доплаты нет.
- Районов без трансфера нет.
- Время забора зависит от экскурсии.

Публичная формулировка:

`Трансфер зависит от района отеля и подтверждается перед бронированием.`

### Гид

- Русскоязычный гид есть на всех экскурсиях.
- Англоязычный гид также есть.
- Русский гид доступен всегда.
- Экскурсий без гида, только с сопровождающим, нет.

Публичная формулировка:

`По данным партнера, экскурсия проводится с русскоязычным гидом. Перед бронированием подтверждаем язык сопровождения по выбранной дате.`

### Страховка

- Страховка включена везде.
- Для страховки нужен паспорт или фото паспорта.
- Возрастных ограничений по страховке нет.
- Страховка есть на активностях: дайвинг, гидроциклы, квадроциклы, рафтинг, зиплайн.

Публичная формулировка:

`Страховка включена. Детали покрытия и документы подтверждаются перед бронированием.`

### Погода, отмена и перенос

- Срок отмены без штрафа уточняется лично при бронировании.
- При плохой погоде возможен перенос или возврат.
- Решение о переносе принимает оператор.
- При отмене из-за погоды деньги возвращаются.
- Перенос даты без штрафа нужно уточнить.
- При сильном опоздании на трансфер возможна отмена без возврата.

Публичная формулировка:

`Отмена и перенос зависят от условий конкретной экскурсии. При плохой погоде оператор может перенести поездку или оформить возврат.`

### Фото

- Фото из Telegra.ph можно использовать на сайте Phuket Go.
- Фото можно использовать в рекламе и соцсетях.
- Оригиналов фото в хорошем качестве нет.
- Можно использовать все фото.
- Сначала фото храним локально.
- Позже можем перенести фото в Cloudinary или другое внешнее хранилище.
- Нейросетевые изображения можно использовать для улучшения или генерации визуалов, но их нужно отмечать как `ai_enhanced` или `ai_generated`.

Открыто:

- не показывать пользователю внутренние ссылки вида `/file/...`.

AI-изображения нельзя выдавать за реальное фото конкретной экскурсии. Они могут быть атмосферной иллюстрацией, но не доказательством маршрута, лодки, гида, отеля или условий.

## Что значит `готова к публикации`

Карточка готова к публикации, если:

- название понятно пользователю;
- категория выбрана корректно;
- цена `от` не является случайной опцией или доплатой;
- цена для пользователя будет пересчитана в рубли;
- в карточке есть предупреждение о курсе;
- программа не обещает неподтвержденные условия;
- трансфер описан осторожно;
- гид и страховка описаны по ответам партнера;
- слоны, если есть, только без катания;
- ограничения вынесены в понятный блок;
- есть текст `кому подойдет`;
- есть текст `кому может не подойти`;
- есть SEO URL, H1, title, description;
- есть список вопросов, которые еще нужно уточнить;
- есть статус публикации.
