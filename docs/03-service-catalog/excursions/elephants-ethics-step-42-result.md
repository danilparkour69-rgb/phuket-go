# Step 42 Result: Проверка слонов и этики

Статус: [x] done для планирования MVP

Дата: 2026-06-28

## Цель

Зафиксировать правило Phuket Go по программам со слонами:

- что можно продавать;
- что нельзя продавать;
- какие карточки требуют контроля;
- какие формулировки и фото нельзя выводить на сайт.

## Главное решение

Phuket Go не продает катание на слонах.

Если в программе есть слоны, публично оставляем только:

- кормление;
- наблюдение;
- купание, если партнер подтверждает формат;
- уход;
- слоновье SPA без катания;
- другой формат без катания, если он подтвержден партнером.

## Что запрещено публиковать

Нельзя выводить на сайт:

- предложения катания на слонах;
- фото, где люди сидят на слонах;
- тексты про прогулку верхом на слоне;
- формулировки, которые создают ожидание катания;
- программы, где формат без катания не подтвержден.

Если программа не может быть заменена на формат без катания, карточку не публикуем в MVP.

## Безопасные публичные формулировки

Можно писать:

```text
Phuket Go не продает катание на слонах. Если программа включает слонов, оставляем только кормление, купание, уход или наблюдение без катания.
```

Можно писать:

```text
Формат со слонами подтверждается перед бронированием. В Phuket Go доступны только программы без катания.
```

Можно писать:

```text
Возможна встреча со слонами в формате кормления, наблюдения или ухода, если такой вариант подтвержден партнером.
```

Нельзя писать:

```text
Прокатитесь на слонах.
```

Нельзя писать:

```text
Катание на слонах входит в программу.
```

## Особые замечания по карточкам

### 037 Каолак

В карточке есть исходная формулировка про поездку, где туристы "прокатятся на слонах".

Решение:

- эту формулировку нельзя выводить на сайт;
- карточка требует переписывания публичного текста;
- если партнер не подтверждает формат без катания, карточку не публикуем.

### 035 Парк слонов

Карточка подходит только в формате:

- без катания;
- кормление;
- наблюдение;
- уход;
- купание, если подтверждено партнером.

Цена требует отдельной проверки по пункту `40`.

### 039 Рафтинг

Допустимо писать только:

- рафтинг;
- квадроциклы, если входят в выбранный вариант;
- зиплайн, если входит в выбранный вариант;
- слоновье SPA без катания, если партнер подтверждает.

## Правила для фото

Нельзя использовать:

- фото с катанием на слонах;
- фото с седлами или конструкциями для катания;
- фото, где турист сидит на слоне;
- фото, которые создают ощущение эксплуатации животного.

Можно использовать:

- кормление;
- наблюдение;
- купание, если это подтвержденная часть программы;
- уход;
- парк слонов без катания;
- спокойные кадры рядом со слонами.

Если фото временное или AI, оно должно быть помечено как временное и требующее проверки.

## Правила для SEO и статей

Можно использовать запросы:

- `парк слонов Пхукет без катания`;
- `слоны Пхукет без катания`;
- `этичная встреча со слонами Пхукет`;
- `кормление слонов Пхукет`.

Нельзя продвигаться под обещание катания.

Если пользователь ищет катание на слонах, Phuket Go должен объяснять, что этот формат не продается, и предлагать альтернативу без катания.

## Решение для MVP

В MVP карточки со слонами можно публиковать только после проверки:

- текст не обещает катание;
- программа без катания подтверждена;
- фото без катания;
- цена проверена;
- условия безопасности и ограничения уточняются перед бронированием.

Если хотя бы один пункт не выполнен, карточка остается в статусе `needs_partner` или `needs_media`.
