# Карта данных экскурсий для будущего сайта

Статус: [x] рабочая база подготовлена

Кодинг проекта пока не начинаем.

Этот документ объясняет файл `excursions-site-data-mvp.json` простым языком. Это не база данных сайта, а подготовленная рабочая таблица в формате, который потом можно будет перенести в сайт без ручной сборки карточек.

## Где лежит база

Файл:

- `docs/03-service-catalog/excursions-site-data-mvp.json`

Источники:

- карточки экскурсий: `docs/03-service-catalog/excursions/`;
- публичные тексты: `docs/07-seo-and-content/excursion-pages-copy-mvp.md`;
- фотографии: `docs/03-service-catalog/media/excursions/`;
- список фотографий: `docs/03-service-catalog/media/excursions/all-excursions-media-manifest.json`.

## Что внутри

В базе сейчас:

- 54 экскурсии;
- у каждой экскурсии есть отдельный `id`;
- у каждой экскурсии есть адрес будущей страницы;
- у каждой экскурсии есть SEO-заголовок и SEO-описание;
- у каждой экскурсии есть цена в THB для внутренней базы;
- публичная валюта зафиксирована как RUB;
- у каждой экскурсии есть папка с фотографиями;
- у каждой экскурсии есть обложка и 3 фото галереи;
- у каждой экскурсии стоит пометка, что фото нужно визуально проверить перед публикацией.

## Главные поля

| Поле | Что значит |
| --- | --- |
| `id` | Номер экскурсии в нашей базе |
| `slug` | Техническое короткое имя экскурсии |
| `title` | Название экскурсии |
| `selling_title` | Продающее название для карточки на сайте |
| `city` | Город, сейчас Phuket |
| `country` | Страна, сейчас Thailand |
| `language` | Язык контента, сейчас `ru` |
| `service_category` | Категория услуги, сейчас `excursion` |
| `source_url` | Ссылка на исходное описание |
| `public_url` | Будущий адрес страницы на сайте |
| `h1` | Главный заголовок страницы |
| `seo_title` | Заголовок для поисковиков |
| `seo_description` | Описание для поисковиков |
| `price_from_thb` | Цена от в батах, для внутренней базы |
| `public_currency` | Публичная валюта, RUB |
| `currency_note` | Предупреждение о курсе рубля |
| `commission_rule` | Правило комиссии |
| `media` | Фотографии экскурсии |
| `flags` | Важные правила карточки |

## Важные правила

- На сайте оплаты нет.
- Пользователь оставляет заявку.
- Оплата происходит на стороне партнера.
- Партнер публично не показывается.
- Мы продаем не экскурсию, а мечту, эмоции и воспоминания.
- Для карточки используем два названия: техническое `title` и продающее `selling_title`.
- Перед публикацией все публичные названия должны быть полностью на русском языке: `title`, `selling_title`, `h1`, `seo_title` и видимые подписи в карточках. Латиница остается только в технических полях вроде `slug`, ID, имен файлов и внутренних ключей.
- Цены на сайте показываем в рублях с пометкой, что из-за курса сумма может отличаться.
- THB оставляем как внутреннюю валюту для партнера и комиссии.
- Дайвинг в будущем отправляется отдельному исполнителю.
- Катание на слонах не продаем.
- Для слонов допускаем только этичные форматы: кормление, наблюдение, купание, парк без катания.

## Что нужно проверить перед публикацией

- Две экскурсии требуют отдельной проверки цены:
  - `020 Пхи-Пхи 2 дня`;
  - `022 5 Жемчужин`.
- Все фотографии нужно визуально проверить:
  - убрать повторы;
  - выбрать более яркие и живые обложки;
  - заменить временные фото, если они не подходят;
  - отдельно проверить фото со слонами, чтобы там не было катания.
- Перед запуском уточнить у партнера актуальность цен.
- Перед запуском перенести продающие названия из `excursions/title-check-step-44-result.md` в рабочие данные сайта.
- Перед запуском проверить, что названия экскурсий, H1, SEO title и карточки каталога полностью русскоязычные.
