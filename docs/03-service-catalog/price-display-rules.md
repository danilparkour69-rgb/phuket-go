# Price Display Rules

Статус: [x] правило MVP зафиксировано

## Решение

Для MVP Phuket Go публично показывает цену экскурсии в батах крупно.

Для русскоязычной аудитории рядом мелко показываем расчетную цену в рублях.

Для будущей англоязычной версии рядом мелко показываем расчетную цену в долларах.

Внутри базы сохраняем исходную цену в THB, потому что:

- партнеры считают цены в батах;
- комиссия Phuket Go считается в THB;
- при изменении курса рубля или доллара публичную расчетную цену нужно пересчитывать, а не менять исходную цену услуги.

## Как показываем цену пользователю

На карточке экскурсии:

`от 3 500 ฿`

Для русской версии мелким текстом:

`примерно 8 500 ₽ по текущему курсу`

Для будущей английской версии мелким текстом:

`about $95 at the current exchange rate`

На странице экскурсии:

- основная цена: в батах;
- для русской версии рядом или ниже мелким текстом: расчетная цена в рублях;
- для будущей английской версии рядом или ниже мелким текстом: расчетная цена в долларах;
- в условиях бронирования: оплата проходит не на сайте Phuket Go, а по правилам партнера.

## Что хранить в базе

| Поле | Что означает |
| --- | --- |
| Цена от THB | Исходная цена партнера |
| Валюта партнера | THB |
| Цена от RUB | Расчетная публичная цена |
| Цена от USD | Расчетная публичная цена для будущей английской версии |
| Курс THB/RUB | Курс, по которому рассчитали рублевую цену |
| Курс THB/USD | Курс, по которому рассчитали долларовую цену |
| Дата курса | Когда был обновлен курс |
| Текст предупреждения | Текст про изменение курса |

## Правило пересчета

Пока на этапе планирования:

- курс не фиксируем;
- точный источник курса выберем перед публикацией;
- цена в рублях должна обновляться при обновлении курса.
- цена в долларах понадобится для будущей английской версии и должна обновляться по тому же принципу.

Перед запуском нужно решить:

- откуда брать курс THB/RUB;
- откуда брать курс THB/USD для будущей английской версии;
- как часто обновлять курс;
- округлять расчетную цену в рублях до 10, 50 или 100 рублей;
- округлять расчетную цену в долларах для английской версии.

## Важно для текстов

Не писать "точная цена в рублях" или "точная цена в долларах", если цена зависит от курса.

Правильная формулировка:

`от 3 500 ฿`

`примерно 8 500 ₽ по текущему курсу`
