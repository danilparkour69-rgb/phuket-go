# Media Storage Task

Статус: [~] draft

## Задача

Определить, где хранить фотографии экскурсий.

## Что нужно решить

- где хранить фото;
- как загружать фото в админке;
- сколько фото можно добавить к одной экскурсии;
- какое фото является обложкой;
- нужны ли разные размеры фото для сайта и Telegram Mini App;
- как переносить фото при переходе на полноценную архитектуру.

## MVP

В первом MVP достаточно фотогалереи.

Видео в первом MVP не нужно.

## Решение для MVP

Фотографии экскурсий храним в Cloudinary.

Причина: популярный сервис для хранения и обработки изображений, есть бесплатный стартовый вариант.

## Future

В будущем можно рассмотреть Yandex Object Storage, если проект будет полностью размещаться в Yandex Cloud.
