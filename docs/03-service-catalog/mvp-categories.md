# MVP Categories

Статус: [~] draft

## Входит в MVP

- Экскурсии.

Внутри экскурсий могут быть:

- острова;
- морские направления;
- прогулки по Пхукету;
- храмы;
- обзорные маршруты;
- другие экскурсионные направления.

## Не входит в MVP

- Самостоятельное добавление карточек партнерами.
- Полноценный открытый маркетплейс.
- Аренда байков.
- Аренда автомобилей.
- Трансферы.
- Обмен денег.
- Недвижимость.
- Визы.
- Документы.
- SIM-карты.
- Медицина.
- Другие неэкскурсионные категории.

## Почему выбраны эти категории

Первая аудитория - русскоязычные туристы. Для них важнее всего быстро найти надежные туристические услуги в одном месте и оставить заявку без поиска по разрозненным Telegram-постам и чатам.

MVP должен сначала проверить спрос на экскурсии и обработку заявок.
