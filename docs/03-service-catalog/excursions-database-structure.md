# Excursions Database Structure

Статус: [~] draft

## Задача

Описать структуру базы экскурсий для MVP.

Одна строка = одна экскурсия.

## Основные колонки

| Колонка | Что означает |
| --- | --- |
| ID | Уникальный номер экскурсии |
| Название | Название экскурсии |
| Направление | Морская, наземная, шоу, активность, премиум и т.д. |
| Короткое описание | Краткое описание для карточки |
| Полное описание | Текст для страницы экскурсии |
| Источник | Ссылка на исходный Telegram/Telegra.ph материал |
| Цена от THB | Исходная минимальная цена партнера |
| Валюта партнера | THB |
| Цена от RUB | Расчетная цена для русской версии |
| Цена от USD | Расчетная цена для будущей английской версии |
| Курс THB/RUB | Курс, по которому рассчитана цена в рублях |
| Курс THB/USD | Курс, по которому рассчитана цена в долларах |
| Дата курса | Когда обновляли курс |
| Примечание по курсу | Текст о том, что RUB/USD являются расчетными ориентирами |
| Варианты | Если есть разные программы/тарифы |
| Длительность | Сколько длится экскурсия; если есть расписание, считаем по первому и последнему времени программы |
| Источник длительности | Рассчитано из расписания / указано явно / зависит от варианта / нужно уточнить |
| Маршрут | Какие места посещает человек |
| Что включено | Трансфер, питание, билеты, гид, страховка и т.д. |
| Что взять с собой | Одежда, купальник, полотенце, деньги и т.д. |
| Страховка | Есть/нет/уточнить |
| Русскоязычный гид | Есть/нет/уточнить |
| Условия отмены | Правила отмены |
| Фото | Ссылки на фото |
| Обложка | Главное фото |
| Отзывы | Текстовые отзывы, добавляет админ |
| Партнер | Внутреннее поле, пользователю не показывается |
| Комиссия | Комиссия Phuket Go |
| Статус | Черновик, опубликовано, скрыто |
| SEO title | Заголовок для поиска |
| SEO description | Описание для поиска |
| URL | Адрес страницы экскурсии |

## Что заполняем из Telegra.ph

- название;
- описание;
- программу;
- маршрут;
- цены;
- варианты;
- фото;
- что включено;
- условия;
- длительность, если указана явно;
- расписание дня, если по нему можно рассчитать длительность.

## Что переписываем

- короткое описание;
- полное описание;
- SEO title;
- SEO description;
- блоки, которые должны продавать идею поездки.

## Что уточняем у партнера

- актуальная цена;
- комиссия;
- страховка;
- русскоязычный гид;
- условия отмены;
- доступные даты;
- что включено;
- что оплачивается отдельно.

Длительность у партнера спрашиваем только если в источнике нет расписания с началом и финалом. Если расписание есть, сначала считаем длительность сами: первое время программы -> последнее время программы. Для диапазона забора из отелей берем раннюю границу как начало окна и отмечаем, что фактический трансфер зависит от района.

## Важно

Тексты из источников не копируем.

Используем источники как фактуру, затем переписываем под стиль Phuket Go.

## Правило публичной цены

Пользователю показываем цену в батах крупно.

Для русской версии рядом мелко показываем расчетную цену в рублях.

Для будущей английской версии рядом мелко показываем расчетную цену в долларах.

THB остается основной валютой базы, партнера и оплаты.

Пример для русской версии:

`от 3 500 ฿`

`примерно 8 500 ₽ по текущему курсу`
